CASHEYE! Mata Untuk Setiap Transaksi

Aplikasi mobile berbasis AI dan text-to-speech yang membantu pengguna dengan gangguan penglihatan mengenali uang Rupiah lewat kamera smartphone dan menyuarakan nominalnya secara otomatis. Aplikasi ini mandiri, ramah disabilitas, dan dapat digunakan kapan saja.

Tentang Proyek Casheye

Casheye merevolusi aksesibilitas bagi komunitas tunanetra dan individu dengan gangguan penglihatan di Indonesia.

Latar Belakang Masalah

  • Di Indonesia, banyak penyandang tunanetra, lansia, dan penjual lanjut usia kesulitan mengenali uang.
  • Solusi yang ada masih bergantung pada bantuan orang lain, yang tidak selalu tersedia dan berisiko menimbulkan salah nominal atau penipuan.
  • Keterbatasan teknologi bantu yang tersedia.

Tujuan Utama Kami

  • Membantu Penyandang Tunanetra dan Individu dengan Gangguan Penglihatan.
  • Mengurangi Risiko Penipuan dan Kesalahan Transaksi.
  • Memanfaatkan Teknologi untuk Mewujudkan Inklusi Finansial.
  • Mendukung Lansia dan Individu dengan Gangguan Penglihatan Ringan.

Fitur Unggulan Casheye

Casheye menggabungkan teknologi mutakhir dengan desain intuitif untuk menciptakan pengalaman yang mudah diakses.

Umpan Balik Suara

Memberikan informasi nominal uang dalam bentuk suara (Audio Feedback).

Klasifikasi Otomatis Uang Rupiah

Mendeteksi dan mengenali nominal uang Rupiah secara cepat dan akurat menggunakan model AI YOLOv5.

Penggunaan Kamera Langsung

Memanfaatkan kamera (webcam) perangkat pengguna yang diakses melalui JavaScript untuk menangkap gambar uang secara langsung.

Pilihan Kamera Depan/Belakang

Fleksibilitas untuk menggunakan kamera depan atau belakang smartphone.

Antarmuka Ramah Aksesibilitas

Antarmuka sederhana, ringan, responsif, dan ramah aksesibilitas.

Integrasi Suara Bahasa Indonesia

Hasil deteksi dikonversi menjadi suara menggunakan Google Text-to-Speech (gTTS), lalu diputar.

Kelebihan Sistem

Deteksi Cepat & Real-time

Deteksi cepat dan real-time.

Dapat Digunakan Tanpa Instalasi

Cukup via browser (PWA).

Sudah Mendukung Audio Output

Memberikan umpan balik suara otomatis.

Teknologi yang Digunakan

Detail teknologi inti yang mendukung Casheye beroperasi.

Frontend

  • Framework: CSS dan JS digunakan untuk membangun antarmuka web yang sederhana, ringan, dan responsif.
  • Fungsi: Menampilkan tampilan kamera, hasil klasifikasi uang, dan memutar audio.
  • PWA (Progressive Web App): Aplikasi berbasis web yang dapat diakses langsung melalui browser, dapat di-install ke layar utama, bekerja ringan, dan cocok untuk perangkat dengan keterbatasan.
  • Alasan: Blade mudah digunakan dan terintegrasi langsung dengan Laravel. Dengan PWA, aplikasi dapat digunakan tanpa instalasi tambahan.

Backend

  • Laravel: Mengelola alur data dari kamera ke model AI dan memproses hasil klasifikasi untuk ditampilkan.
  • FastAPI (Python): Berfungsi sebagai microservice inference untuk model YOLOv5 yang mendeteksi nominal uang Rupiah.
  • Komunikasi: Laravel mengirimkan gambar ke FastAPI, lalu menerima label deteksi dan confidence score.
  • Alasan: Laravel kuat dalam manajemen web dan routing, sementara FastAPI cepat dan ringan untuk inferensi model AI.

Model AI & Dataset

  • Jenis Model: YOLOv5s.
  • Tipe Task: Object Detection.
  • Arsitektur: Convolutional Neural Network (CNN) berbasis YOLOv5s.
  • Framework: PyTorch melalui implementasi resmi YOLOv5.
  • Jumlah Kelas: 7 kelas uang rupiah (1k, 2k, 5k, 10k, 20k, 50k, 100k).
  • Jumlah Data: 1.200 gambar.
  • Sumber Dataset: Koleksi pribadi.
  • Tool Anotasi: Label Studio.
  • Format Output: YOLO format (bounding box & label kelas).
  • Pembagian Data: 80% pelatihan, 20% validasi.
  • Proses Pelatihan: Dilatih menggunakan PyTorch dengan YOLOv5 selama 40 epoch. Menggunakan optimizer SGD atau Adam.
  • Augmentasi Data: Dilakukan otomatis selama pelatihan untuk meningkatkan generalisasi model.
